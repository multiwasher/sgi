🚀 PRÓXIMOS PASSOS (3 PASSOS SIMPLES)
──────────────────────────────────────────────────────────────────────────────

PASSO 1: Criar o Leitor de Dados
  1. Abra: https://docs.google.com/spreadsheets/d/1D_lN4o_bRWBNMDQwOue7TdkrZZQ6z5hQjl48A3EH_jc
  2. Extensões → Apps Script
  3. Cole o código de: plano-acao.gs
  4. Guarde

PASSO 2: Publicar como Web App
  1. Deploy → New deployment
  2. Tipo: Web app
  3. Execute as: seu email
  4. Who has access: Anyone
  5. Deploy
  6. Autorize
  7. COPIE A URL

PASSO 3: Atualizar index.html
  1. Procure: PLANO_ACAO_APPS_SCRIPT_URL
  2. Substitua: SUBSTITUIR_COM_URL_REAL
  3. Cole a URL do Passo 2
  4. Guarde

✅ PRONTO!

  Recarregue o portal e clique em "PLANO DE AÇÃO" ✨
